Sobre este projeto e seu desenvolvimento !

Este projeto se baseia em uma idéia simples com o intuito de ser um sistema Web que possibilite os cadastros de rendimentos e despesas para que disponibilizar um maior controle das finanças do dia a dia.

No atual momento é uma idéia bem simples, armazenando os cadastros feito pelo usuário apenas em (Local Storage), com o objetivo de levar ao próximo nível as técnicas de desenvolvimento, trazendo como base de desenvolvimento de um sitema Web as tecnologias HTML, CSS, Java Script e também Bootstrap, sendo o seu desenvolviemnto em grande parte orientado a objeto, juntamente com alguns metodos do AJAX.
